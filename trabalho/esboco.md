# TRABALHO DE EQUAÇÕES DIFERENCIAIS ORDINÁRIAS
Este trabalho propõe o desenvolvimento de uma modelagem envolvendo equações diferenciais ordinárias não-homogêneas. O grupo deve escolher _ou_ desenvolver uma modelagem, justificar sua relevância e aplicação e realizar seu estudo qualitativo.

## Entregas

As entregas do trabalho serão dois relatórios em formato _.pdf_, sendo o primeiro uma entrega parcial (_Primeira Etapa_), e o segundo a entrega final (_Primeira Etapa_ acrescida da _Segunda Etapa_). O conteúdo dos relatórios é apresentado abaixo:

### Conteúdo Geral
* Capa
* Introdução
* Desenvolvimento matemático do modelo
    * Desenvolvimento literal
    * Análise gráfica do modelo considerando diferentes casos

### Primeira Etapa
Nessa etapa, deve ser apresentada uma EDO não-homogênea de ordem _n_.
